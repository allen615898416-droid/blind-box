## Page 3 — Game（问题最多，逐条列）

### 🔴 必须删除：
- **删除 "HIDDEN EDITION!" 标签** — 游戏页不能暴露哪个是隐藏款，盒子必须完全相同外观

### 🔴 必须新增：
**❶ 目标角色区（页面顶部）**
```
find:  [轮廓?]  (0/1)
```
- 正常状态：1个角色轮廓剪影 + "?" + 计数器(0/1)
- 隐藏款触发时：2个目标并列
```
find:  [普通?]  [⭐隐藏?]  (0/1)
```
- 用轮廓剪影区分，不要大字"HIDDEN EDITION"

**❷ 系列切换 Tab 栏**
- 同Page2，游戏页也要能切换系列

### 🟡 调整：
**❸ "Your pick" → 改用按钮文案解决**
- 不需要额外视觉标签
- CONFIRM按钮文案改为 **"Pick it"**
- 一句话说明：你按的就是中间那个盒子

**❹ 三个盒子必须完全相同**
- 同颜色、同图案、同"?"
- 只有大小不同（中间大，两边小+半透明）
- 不能有任何颜色/装饰差异
